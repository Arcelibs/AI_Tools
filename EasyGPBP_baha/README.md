# BetterBaha GP/BP

一個給巴哈姆特哈啦板列表頁使用的 Tampermonkey userscript

它會在文章列表左側的 GP 區塊補上 BP 數量，方便快速查看文章的 GP / BP 狀態，不需要逐篇點進文章

## 功能

直接在前面顯示GP跟BP

## 用法

使用Tampermonkey將 `betterbaha-gpbp.user.js` 的內容貼上並儲存，開啟巴哈姆特哈啦板列表頁就好
