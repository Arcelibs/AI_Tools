# officemode_baha 上班模式說明

`betterbaha-work-mode.user.js` 是給巴哈姆特哈啦板使用的 Tampermonkey / Violentmonkey 腳本

它的目標很單純：讓巴哈頁面在上班或公共場合瀏覽時低調一點，預設收合右側聊天區，並把容易引人注意的圖片先藏起來

## 功能

### 1. 自動收合 Bahamut NOW

右側的 Bahamut NOW 聊天區會自動收合成一條可展開的區塊

### 2. 預設隱藏列表頁圖片

文章列表中的縮圖會先變成「圖片已隱藏」遮罩。

目前包含：

- 文章列表縮圖
- 頁面下方「本板推薦」卡片圖片

### 3. 預設隱藏文章頁圖片

文章閱讀頁中的圖片會先變成小型遮罩，點一下才顯示。

目前包含：

- 文章內文圖片
- 留言中的圖片
- 文章頁下方「延伸閱讀」卡片圖片
